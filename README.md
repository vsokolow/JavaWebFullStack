# JavaWebFullStack

Учебный репозиторий с практическими заданиями и материалами занятий по 
курсу **Java Web Full Stack**. Здесь собраны фронтенд-практики: работа с 
HTML, CSS и vanilla JavaScript, выполненные в рамках занятий курса.

> ⚠️ Это учебный архив, а не готовый портфолио-проект. Материалы 
организованы по датам занятий и могут содержать черновой, незавершённый 
или экспериментальный код.

## Стек технологий

- **JavaScript** — основная часть заданий (~70%)
- **CSS** — стилизация (~19%)
- **HTML** — разметка (~11%)

## Структура репозитория

Каждая папка соответствует отдельному занятию и названа по дате его 
проведения (`lecture_ДД_ММ`)

## Как запустить

Большинство заданий — это отдельные HTML-страницы с подключёнными CSS и 
JS. Чтобы посмотреть конкретное занятие:

```bash
git clone https://github.com/vsokolow/JavaWebFullStack.git
cd JavaWebFullStack/lecture_XX_XX
```

Затем открыть `index.html` (или другой основной HTML-файл папки) в 
браузере.

## Статус

Репозиторий пополняется по мере прохождения курса. Отдельные пет-проекты 
на основе полученных навыков вынесены в отдельные репозитории.
